# Repositorio de Flujos n8n

Este repositorio contiene distintos flujos de trabajo de **n8n**, cada uno con su JSON y documentación detallada.

---

## Flujos disponibles

- [**Flujo 1: Quote of the day Slack**]  
  Envía una cita diaria a Slack y se conecta a un agente para añadir más información sobre la cita.
  
- [**Flujo 2: OpenWeather AI Agent **]  
  Envía un mensaje a Slack sobre el clima de una ciudad o pueblo, con noticias del día y un dato curioso de Wikipedia.
